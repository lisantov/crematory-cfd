# Проект "Крематорий"
SPA сайт предоставляющий услуги кремации с личным кабинетом, отзывами, заказами и каталогом

------------------------------------------

## Стек проекта
### Front:
  - HTML5
  - CSS/SCSS
  - TypeScript
  - Vue 3 Composition API
  - Pinia
  - Vue Router
  - Vite
  - ESLint + Prettier

### Back:
  - PHP >= v8.2
  - Laravel v12.x
  - PostgreSQL
  - Docker + Docker compose
  - Postman
  - Nginx

------------------------------------------
## Запуск проекта 

Для начала откройте Docker Desktop на вашем ПК.
Перейдите в папку ./deploy и напишите команду docker-compose up -d --build 

После установки всех зависимостей и окончания билда докера, проект будет запущен и готов к работе.

Проверить работу контейнеров можно воспользовавшись коллекцией Postman для бэкенда,
либо перейдя по адресу http://localhost/ для проверки фронта.
