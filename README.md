# Проект "Крематорий"
SPA сайт предоставляющий услуги кремации с личным кабинетом, отзывами, заказами и каталогом

------------------------------------------

## Стек проекта
### Front:
  - HTML5
  - CSS/SCSS
  - TypeScript
  - Vue 3 Composition API
  - Pinia
  - Vue Router
  - Vite
  - ESLint + Prettier

### Back:
  - PHP >= v8.2
  - Laravel v12.x
  - PostgreSQL
  - TypeScript
  - Docker + Docker compose
  - Nginx
